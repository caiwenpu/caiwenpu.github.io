ImageNet

1,200,000 训练集 50,000个验证集 150,000个测试集 1000个类别
在Alexnet上 5.2ms 一个input,包括forward和backward

训练一轮大概需要1.733h
所以一般在pretrained的网络，调整网络结构，使用新的数据集，进行retrain
Alexnet 训练90轮 :

直接从头开始pretrain不现实

数据加载

  • 顺序加载，局部shuffle
  • cv2.imdecode 图片解码
  • origin data:tensorpack 的 loader 第一个epoch很慢，之后很快(因为前边部分已经加载到了内存中)

数据预处理

  • resize的方法：
    • google resize，随机裁剪为224x224 (RandomSizedCrop(224))
    • resize 短边为256，随机裁剪为224x224
    • resize 为256x256，随机裁剪为224x224
    • 随机裁剪增加噪声防止过拟合
  • channel变换(很少用)
  • 归一化到[0,1],格式转变为NCHW (这种格式计算效率更高)
  • The mean/std processing 可以提高训练结果.
  • 备注:Note that Caffe LMDB format is not efficient: it stores serialized raw arrays rather than JPEG images.
