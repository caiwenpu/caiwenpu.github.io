ImageNet

1,200,000 训练集 50,000个验证集 150,000个测试集 1000个类别
在Alexnet上 5.2ms 一个input,包括forward和backward

训练一轮大概需要1.733h
所以一般在pretrained的网络，调整网络结构，使用新的数据集，进行retrain
Alexnet 训练90轮 :

直接从头开始pretrain不现实

数据加载

  • 顺序加载，局部shuffle
  • cv2.imdecode 图片解码
  • origin data:tensorpack 的 loader 第一个epoch很慢，之后很快(因为前边部分已经加载到了内存中)

数据预处理

  • resize的方法：
    • google resize，随机裁剪为224x224 (RandomSizedCrop(224))
    • resize 短边为256，随机裁剪为224x224
    • resize 为256x256，随机裁剪为224x224
    • 随机裁剪增加噪声防止过拟合
  • channel变换(很少用)
  • 归一化到[0,1],格式转变为NCHW (这种格式计算效率更高)
  • The mean/std processing 可以提高训练结果.
  • 备注:Note that Caffe LMDB format is not efficient: it stores serialized raw arrays rather than JPEG images.

Quantized Neural Network

low-precision Quantization

quantized weights only

每个float权重近似表示成几个bits

  • Matthieu-Courbariaux,Yoshua-Bengio,Jean-Pierre-David:“BinaryConnect: Training Deep Neural Networks with binary weights during propagations.” [NIPS 2015]

  • Jiaxiang Wu, Cong Leng, Yuhang Wang, Qinghao Hu, Jian Cheng:Quantized Convolutional Neural Networks for Mobile Devices[CVPR 2016]

    • 量化卷积层的权重，调整全连接层的全精度权重；再量化全连接层的权重
    • 提供了实际加速的效果对比，3-4倍左右的加速
    • 提供了移动端的实验结果对比，包括内存开销，储存时间，储存开销

  • Chenzhuo Zhu, Song Han, Huizi Mao, William J. Dally:
    Trained Ternary Quantization[ICLR 2017]

  • Aojun Zhou, Anbang Yao, Yiwen Guo, Lin Xu, Yurong Chen:
    “Incremental Network Quantization: Towards Lossless CNNs with Low-Precision Weights.”[ICLR 2017]

    • 简称INQ
    • 大的权重量化为2的幂次，小的权重保持全精度
    • 53$\times$压缩率，结合DNS

  • Yiwen Guo, Anbang Yao, Hao Zhao, Yurong Chen:Network Sketching: Exploiting Binary Structure in Deep CNNs[CVPR 2017]

    • 剩余量化权重
    • 利用Bit平面之间的差值来减少冗余计算
    • 3bit 权重 Resnet18 1% top1精度损失 fine-tuning

  • Felix Juefei-Xu, Vishnu Naresh Boddeti, Marios Savvides:
    Local Binary Convolutional Neural Networks. [CVPR 2017]

    • 一个三值卷积层(固定参数) + 一个1*1卷积层(科学系参数)替代一个完整卷积层
    • 更少的可训练参数，防止过拟合

  • Training Quantized Nets: A Deeper Understanding [NIPS 2017]

  • Learning Accurate Low-Bit Deep Neural Networks with Stochastic Quantization [BMVC 2017]

  • Loss-Aware Weight Quantization of Deep Networks. [ICLR 2018]

    • 基于近端牛顿法，对带有离散权重约束的目标进行优化
    • 优化过程中依然保留浮点权重，权重更新在浮点权重上进行

  • LEARNING DISCRETE WEIGHTS USING THE LOCAL
    REPARAMETERIZATION TRICK[ICLR 2018]

    • 每个权重是一个随机变量，由一组参数刻画。
    • 由中心极限定理，这一层的输出服从正太分布

  • Learning Accurate Low-Bit Deep Neural Networks with Stochastic Quantization. [BMVC 2018]

  • Explicit Loss-Error-Aware Quantization for Low-Bit Deep Neural Networks [CVPR 2018]

  • An Optimal Control Approach to Deep Learning and
    Applications to Discrete-Weight Neural Networks[ICML 2018]

    • 理论依据的离散权重神经网络的训练。

  • From Hashing to CNNs: Training Binary Weight Networks via Hashing. [AAAI 2018]

  • Deep Neural Network Compression with Single and Multiple Level Quantization[AAAI 2018]

    - 增量聚类量化，对权重进行聚类，基于每一个类簇的聚类(量化)对损失函数的影响，把所有的类簇划分量化部分和非量化部分，量化损失函数大的量化部分，重新训练剩余的非量化部分
- 不再每层单独考虑，所有层一起考虑，对所有层的聚类量化损失在一起比较。
- 效果略好于INQ，不是很明显。

  • Cong Leng, Zesheng Dou, Hao Li, Shenghuo Zhu, Rong Jin:Extremely Low Bit Neural Network: Squeeze the Last Bit Out with ADMM[AAAI 2018]

    • 使用ADMM，优化损失函数，更新W
    • 只量化权重到 $\alpha_i {0,1,-1,2,-2…2^N,-2^N}$
    • 在Resnet VGG Googlenet 实验，效果略好于TTQ

  • PROXQUANT: QUANTIZED NEURAL NETWORKS VIA
    PROXIMAL OPERATORS [ICLR 2019]

  • LEARNING RECURRENT BINARY/TERNARY WEIGHTS [ICLR 2019]

  • Add BatchNorm to LSTM for Quantized weights

  • Projection Convolutional Neural Networks for 1-bit CNNs via Discrete Back Propagation [AAAI 2019]

  • Simultaneously Optimizing Weight and Quantizer of Ternary Neural Network using Truncated Gaussian Approximation [CVPR 2019]

  • 连续是定义在某个点上，一致连续是定义在整个函数定义域。
  • Continuously differentiable ⊆ Lipschitz continuous ⊆ absolutely continuous ⊆ bounded variation ⊆ differentiable almost everywhere
  • Hessian矩阵是正定的，一阶导数为0的点为极小值；Hessian矩阵是负定的，一阶导数为0的点为极大值
